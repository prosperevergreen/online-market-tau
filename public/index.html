Front Page

Register | List Users | List Products | Shopping Cart | View orders

This is an assignment for the course Web development 1 - Programming.

This web app has webshop like features. Users can register, view products, add them to cart and order them. After ordering users may view their orders.

Some of the users have Admin credentials and they may view all users information, modify and delete users and view all of the orders in the system.
